# React Photo Gallery App

A simple and responsive Photo Gallery built using React.  
This project fetches photos from an API and displays them in a clean gallery layout with search and favourite features.

## Features

- Fetch photos from an external API
- Display photos in a responsive grid layout
- Real-time search filter by author name
- Mark photos as favourites
- State management using `useReducer`
- Loading and error handling
- Modern UI using Tailwind CSS

## Technologies Used

- React
- JavaScript (ES6+)
- Tailwind CSS
- Vite
- React Hooks (`useState`, `useReducer`, `useMemo`, `useCallback`)
